## Features

- Authentication:
    - [x] Sign in
    - [ ] Sign up
    - [ ] OTP 
    - [ ] Resend OTP
    - [ ] Change password
- Post:
    - [ ] List post
    - [ ] Create post
    - [ ] Upload images (related to creating a post)
    - [ ] Add comment
    - [ ] Update comment
    - [ ] Delete comment
    - [ ] Sort comment by date
    - [ ] Like post
    - [ ] Share post
    - [ ] Unlike post
- User:
    - [ ] Show profile
    - [ ] Update profile
    - [ ] Unfollow user
    - [ ] Follow user
    - [ ] List follower
    - [ ] List following
- Pet:
    - [ ] Create new pet
    - [ ] Delete pet
    - [ ] Update pet
    - [ ] Search
